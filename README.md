# workstation-bootstrap

Bootstrap repository for restoring my Arch/CachyOS workstation.

## Stage 1

Current stage installs software only.

It does not yet restore application settings, dotfiles, browser profiles, VPN profiles, tokens, secrets, or private data.

## Supported system

- CachyOS
- Arch Linux
- Arch-like distributions with pacman

## Install

```bash
git clone git@github.com:USERNAME/workstation-bootstrap.git ~/projects/workstation-bootstrap
cd ~/projects/workstation-bootstrap
./install.sh
